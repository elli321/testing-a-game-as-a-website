# 🌹 Pixel Rose Explorer

A charming 2D pixel game where you explore mysterious locations, collect roses, and discover weird pictures in the gallery.

## Features

- 🎮 **2D Pixel Exploration** - Explore 4 unique locations with different aesthetics
- 🌹 **Rose Collection** - Find and collect roses scattered throughout the world
- 🖼️ **Weird Picture Gallery** - Discover strange procedurally-generated pixel art
- 💬 **Comment System** - Share your thoughts on the weird pictures you find
- 🗺️ **Multiple Worlds** - Travel through Starting Garden, Dark Forest, Abandoned Temple, and Cosmic Void

## How to Play

1. **Open** `index.html` in your web browser
2. **Move** using arrow keys or WASD
3. **Collect Roses** by walking over them
4. **Discover Pictures** as you explore
5. **View Gallery** by pressing G key
6. **Comment** on the weird pictures you find

## Controls

- **Arrow Keys / WASD** - Move your character
- **Space / E** - Interact with objects (collect roses, enter doors, view pictures)
- **G** - Open gallery
- **Ctrl + Enter** - Post comment on a picture

## Game Elements

### Locations
- 🌿 **Starting Garden** - A peaceful garden filled with flowers
- 🌲 **Dark Forest** - A mysterious forest shrouded in shadows
- ⛩️ **Abandoned Temple** - Ancient ruins covered in strange symbols
- 🌌 **Cosmic Void** - A strange realm filled with otherworldly phenomena

### Interactions
- 🌹 **Roses** - Pink circles that you can collect
- 🚪 **Doors** - Golden squares that teleport you to other locations
- 🎨 **Pictures** - Colorful squares that reveal weird pixel art

## Technical Details

Built with:
- HTML5 Canvas for rendering
- Vanilla JavaScript (no dependencies!)
- Local storage for saving progress

## Files

- `index.html` - Main game page
- `style.css` - All styling and layouts
- `game.js` - Core game engine and mechanics
- `gallery.js` - Gallery and commenting system
